# 🌤️ Weather Forecast Application

## Overview

Weather Forecast Application is a responsive web application that provides real-time weather updates for cities worldwide. Users can search for any location and instantly receive accurate weather information, including temperature, humidity, and atmospheric conditions.

The application integrates with the OpenWeather API to fetch live weather data and presents it through a clean, user-friendly interface enhanced with animations and responsive design.

---

## Features

### Weather Information
- Real-time weather updates
- City-based weather search
- Temperature monitoring
- Humidity tracking
- Weather condition display

### User Experience
- Responsive user interface
- Loading indicators
- Error handling for invalid cities
- Smooth animations using AOS
- Fast API-based data retrieval

### API Integration
- OpenWeather API integration
- Real-time data fetching
- Dynamic weather updates
